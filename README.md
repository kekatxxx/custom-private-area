CUSTOM PRIVATE AREA - Francesco Barbieri

LINK

Public repository: https://github.com/kekatxxx/custom-private-area/

DESCRIPTION

A simple Wordpress plugin that creates a private area in a selected page of the website.

ADMIN SETTINGS

Select the page in the "Settings" section that you want to make private.

FRONT-END USAGE

The selected page will display a login/registration form to access on the page. 
Once the login/registration is done, the content of the page is displayed.

Have fun! :)
